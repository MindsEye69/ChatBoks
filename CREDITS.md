# Credits

ChatBoks is a human-supervised multi-agent collaboration tool built around local project context, CodeGraph, and explicit approval gates.

## Workflow Inspirations

Some ChatBoks workflow ideas are inspired by the broader ecosystem of agent workflow cards, coding checklists, and mode-specific prompt patterns. The native skills in this repository are written specifically for ChatBoks and use ChatBoks terminology, signals, and safety boundaries.

No third-party workflow text is copied into the current native skills. If future skills adapt or quote licensed text, that skill file should include the required copyright and license notice.
